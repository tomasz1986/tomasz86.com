About

This website serves as a source of information, updates lists, HFSLIP guidelines, and downloads for Windows 2000, Windows XP and Windows Server 2003.

  • Updates is where you can find current lists of updates for Windows 2000 and XP.
  • HFSLIP provides a basic how-to and a download link for HFSLIP 2000.
  • Archive is a download repository of Windows updates and other related files.
  • Changelog is a history of changes to the website and the updates lists over the course of time.

Before using this page, please read the disclaimer.
